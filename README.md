# Where in the World? - Countries API Project

A modern web application that provides comprehensive information about countries worldwide using the [REST Countries API](https://restcountries.com/). This project focuses on high performance, clean UI, and optimized data fetching.

## ✨ Features

- **Global Search**: Find any country instantly with an optimized local search engine.
- **Region Filtering**: Filter countries by their continent (Africa, Americas, Asia, Europe, Oceania).
- **Detailed View**: Access in-depth information about a specific country, including native name, currencies, languages, and more.
- **Dark/Light Mode**: Seamlessly toggle between dark and light themes with persistent settings.
- **Optimized Fetching**: 
  - Home page uses a targeted fields query for better performance.
  - Details page fetches only the specific country needed (not the entire list).
- **Clickable Borders**: View neighboring countries and navigate to their details with a single click.
- **Responsive Design**: Fully functional on mobile, tablet, and desktop.

## 🛠️ Technologies Used

- **JavaScript (ES6+)**: Core logic and state management.
- **Webpack**: Module bundling and local development server.
- **Vanilla CSS**: Custom styling with CSS variables for theming.
- **Babel**: Transpiling for browser compatibility.
- **REST Countries API**: Source of all world data.

## 📦 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
- Node.js installed on your machine.
- npm (Node Package Manager).

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/shokhruzisroilov/webpack-test-countires-project.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm start
   ```
4. Build for production
   ```sh
   npm run build
   ```
